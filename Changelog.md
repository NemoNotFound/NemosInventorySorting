# Changelog v1.20

## Changes
- Refined the item filter layout.
  - Adjusted the default filter box position, width, and height.
  - Adjusted the default filter persistence toggle position.
- Updated filter box appearance.
  - Added custom focused and unfocused filter box textures.
  - Matched the filter text color to Minecraft's default title color.
  - Removed the text shadow for better readability.
- Reduced the filter input length to better fit the updated filter box.
